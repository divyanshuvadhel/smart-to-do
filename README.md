# Smart To-Do

A modern, responsive task management application with dark mode support and local storage persistence.

🔗 **[Live Demo](https://divyanshuvadhel.github.io/smart-to-do/)**

## Features

- **Quick Add**: Instantly add tasks with Enter key
- **Task Management**: Full CRUD operations with detailed task forms
- **Priority System**: High, Medium, Low priority levels with color coding
- **Categories**: Personal, Work, Shopping, Health, Other
- **Statistics Dashboard**: Track total, completed, pending, and overdue tasks
- **Live Search**: Real-time task searching as you type
- **Filtering**: Search and filter by status, category, and priority
- **Dark Mode**: Toggle between light and dark themes
- **Local Storage**: Automatic data persistence
- **Keyboard Shortcuts**: Alt+N to add new tasks

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Tailwind CSS for styling
- **JavaScript** - Vanilla JS for functionality
- **Lucide Icons** - Modern icon library
- **Local Storage** - Data persistence

## Getting Started

1. Clone or download the project
2. Open `index.html` in your browser
3. Start adding tasks!

## Usage

### Adding Tasks
- **Quick Add**: Type in the quick add input and press Enter
- **Detailed Add**: Click the floating + button or use Alt+N
- **Modal Form**: Fill in title, description, priority, category, and due date

### Managing Tasks
- **Complete**: Click the checkbox to mark as done
- **Edit**: Click the edit icon on any task
- **Delete**: Click the delete icon to remove tasks
- **Clear All**: Use the "Clear Completed" button

### Filtering
- Search tasks by title
- Filter by status (All, Pending, Completed, Overdue)
- Filter by category and priority
